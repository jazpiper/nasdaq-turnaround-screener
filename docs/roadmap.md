# Roadmap

## Phase 0, Documentation
- 목표/범위 명확화
- signals와 score 초안 작성
- OpenClaw 연동 방식 결정

## Phase 1, MVP Screener
- project scaffold
- universe loader
- daily price fetch
- BB / RSI / MA 계산
- 후보 필터와 score
- markdown/json report 생성

## Phase 2, Evaluation
- 최근 6~12개월 데이터 기준 샘플 검토
- false positive 유형 정리
- threshold 조정

## Phase 3, OpenClaw Automation
- cron schedule 연결
- daily summary prompt 정리
- 실패 알림 / 로그 규칙 정리

## Phase 4, Productization
- dashboard 또는 report UI
- watchlist/history 저장
- score trend 시각화
- subscription/service 검토
